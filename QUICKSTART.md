# Quick Start Guide - Item Set Links Module

## Installation (5 minutes)

### Step 1: Upload Module
1. Download the `ItemSetLinks.zip` file
2. Extract it
3. Upload the `ItemSetLinks` folder to your Omeka S server
4. Place it in: `modules/ItemSetLinks/`

### Step 2: Install in Omeka S
1. Log in to Omeka S admin panel
2. Go to **Modules** (left sidebar)
3. Find "Item Set Links" in the list
4. Click **Install**
5. Module is now active!

## Usage (2 minutes)

### Add the Block
1. Go to your site â†’ **Pages**
2. Edit the page where you want the block
3. Click **"Add new block"** (panel opens)
4. Scroll and find **"Item Set Links"**
5. Click it to add

### Configure the Block
You'll see two fields:

**Item Set IDs:** (required)
```
1303, 866, 605, 693, 1372, 558
```
Enter your item set IDs separated by commas.

**Description:** (optional)
```
Database of "Music, Muslims and Jews", here you will find all the source materials and research outcomes of this project.
```
Enter description text or leave empty.

### Save
Click **Save** at the bottom of the page.

## That's It!

Your block is now live. It will:
- Display item set titles in the current language
- Generate working links to browse each item set
- Use your existing CSS styling

## Next Steps

### Set Up Translations
1. Go to **Item Sets** in admin
2. Edit each item set
3. Add titles in multiple languages
4. Save

Now when users switch languages, the item set names will translate automatically!
